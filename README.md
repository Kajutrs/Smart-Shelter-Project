* * рЃџрЃўрЃбрЃћрЃарЃљрЃбрЃБрЃарЃљ:** рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃгрЃДрЃљрЃарЃЮрЃћрЃЉрЃў, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃцрЃБрЃарЃфрЃџрЃћрЃЉрЃў (Datasheets) рЃЊрЃљ рЃЎрЃЋрЃџрЃћрЃЋрЃћрЃЉрЃў.
* **­ЪЊќ рЃАрЃљрЃ«рЃћрЃџрЃЏрЃФрЃдрЃЋрЃљрЃюрЃћрЃџрЃЮ:** рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃљ рЃЊрЃљ рЃћрЃЦрЃАрЃърЃџрЃБрЃљрЃбрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.
* ** рЃЎрЃЮрЃЊрЃўрЃА рЃюрЃљрЃгрЃДрЃЋрЃћрЃбрЃћрЃЉрЃў:** рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ.
* ** рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў:** рЃЉрЃЏрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџ рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃа рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ.
