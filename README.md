# Smart-Shelter-Project
ეს პროექტი წარმოადგენს ინტელექტუალურ სისტემას, რომელიც უზრუნველყოფს ცხოველებისთვის კომფორტული და უსაფრთხო გარემოს შექმნას ავტომატური გათბობისა და დისტანციური მონიტორინგის საშუალებით.

#რა კომპონენტებს ვიყენებთ:
ტვინი: STM32L476RG (Nucleo) - მთელი ლოგიკა აქ გადის.

WiFi: ESP8266 (ESP-01) - მონაცემების ინტერნეტში გასაგზავნად.

სენსორები: DHT22 (ტემპერატურის საზომი) და VL53L0X (ToF ლაზერული სენსორი მანძილისთვის).

გათბობა: 12V გამათბობელი ფირა, რომელიც რელეთი იმართება.



როგორ მუშაობს სისტემა:
მთავარი პრინციპი ისაა, რომ ტყუილად არ დაიხარჯოს ენერგია. გათბობა (PA5 პინი) ჩაირთვება მხოლოდ იმ შემთხვევაში, თუ:

ცხოველი ნამდვილად შიგნითაა (სენსორი აჩვენებს 70 სმ-ზე ნაკლებ მანძილს).

გარეთ ცივა (ტემპერატურა 18°C-ზე ნაკლებია).


პროექტის გეგმა (Milestones):
პროექტი 16 კვირაზე გვაქვს გაწერილი (იხილეთ განტის დიაგრამა drawings საქაღალდეში):

დასაწყისი (1-4 კვირა): სქემების დახაზვა და კომპონენტების შერჩევა.

შუა ეტაპი (5-12 კვირა): 3D მოდელის აწყობა და STM32-ის კოდზე მუშაობა.

ფინალი (13-16 კვირა): ყველაფრის ერთად შეკვრა, Go სერვერის გამართვა და ტესტირება.
