## 1. სისტემის მიმოხილვა
ეს სისტემა წარმოადგენს ინტეგრირებულ გადაწყვეტილებას ცხოველთა თავშესაფრისთვის, რომელიც აერთიანებს მიკროკონტროლერულ მართვას (STM32) და პროფესიონალურ ვიდეო მონიტორინგს (**TP-Link Tapo C310**).

### 1.1 კომპონენტების დეტალური ცხრილი
| კომპონენტი | მოდელი | დანიშნულება |
| :--- | :--- | :--- |
| **მიკროკონტროლერი** | STM32L476RG | სენსორების დამუშავება და ლოგიკა |
| **კამერა** | Tapo C310 | ვიდეო მონიტორინგი და ნაკადის გაშვება |
| **ტემპერატურის სენსორი** | DHT22 | კლიმატის კონტროლი |
| **მანძილის სენსორი** | VL53L0X | ცხოველის ყოფნის დაფიქსირება |
| **WiFi მოდული** | ESP-01 | მონაცემთა გადაცემა სერვერზე |



---

## 2. აპარატურული ნაწილი და კავშირები (Pinout)

სენსორების დაერთება Nucleo დაფასთან უნდა მოხდეს შემდეგი სქემით:
* **PA1:** DHT22 (ტემპერატურა და ტენიანობა)
* **PB8 (SCL) / PB9 (SDA):** VL53L0X (I2C პროტოკოლი)
* **PA5:** რელეს მოდული (გამათბობლის მართვა)
* **PA2 (TX) / PA3 (RX):** ESP-01 (UART კომუნიკაცია)
